## Установка и запуск
1. Склонируйте репозиторий и перейдите в папку с проектом 
2. Установите все необходимые зависимости, выполнив команду (предварительно создайте и активируйте виртуальное окружение):
```bash
pip install -r requirements.txt
```
3. Замените TELEGRAM_TOKEN в chatbot.py на свой
4. Запустите, выполнив команду:
```bash
python chatbot.py
```
5. Можно потестить в ТГ [KvintTestOrderPizzaBot](https://t.me/KvintTestOrderPizzaBot)
